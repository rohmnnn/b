<!--more-->

## Langkah-langkah

### 1. Install Executor
Menggunakan emulator Android seperti **BlueStacks** atau **LDPlayer** atau **MuMu Player** lebih aman:

- Kunjungi [delta-executor.com](https://delta-executor.com)
- Download Delta Executor APK untuk Android
- Install di perangkat Android atau emulator
- Lakukan aktivasi dengan key

### 2. Masuk ke Game

- Buka Roblox dan login ke akun Anda (gunakan akun alt jika di emulator)
- Masuk ke game **Fish It**

### 3. Inject dan Jalankan Script

- Buka Delta Executor
- Klik tombol **Inject** atau **Attach**
- Copy dan paste script berikut ke executor:

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/MajestySkie/Chloe-X/main/Main/ChloeX"))()
```

- Klik **Execute** atau **Run**

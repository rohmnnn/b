Auto Boot adalah fitur yang memungkinkan HP Android menyala otomatis saat dicolokkan ke charger, tanpa perlu menekan tombol power.

<!--more-->

## Prasyarat

- Android 10 atau lebih tinggi
- Bootloader sudah di-unlock
- Magisk terinstal

## Download Modul

Download dari GitHub releases:
```
https://github.com/anasfanani/magisk-autoboot/releases
```

Tersedia 2 versi:
- **Simple Version (v2.0.0)** - Hanya fitur auto boot (recommended)
- **Full Version** - Dengan fitur tambahan seperti auto reboot scheduler

## Install via Magisk Manager

1. Buka aplikasi Magisk Manager
2. Pilih menu **Modules**
3. Tap **Install from storage**
4. Pilih file `.zip` modul yang sudah didownload
5. Reboot setelah instalasi selesai
